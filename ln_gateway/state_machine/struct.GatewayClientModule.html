pub async fn remove_from_federation(&self, gateway_keypair: KeyPair)

Attempts to remove a gateway’s registration from the federation. Since removing gateway registrations is best effort, this does not return an error and simply emits a warning when the registration cannot be removed.

Retrieves the signing challenge from each federation peer. Since each peer maintains their own list of registered gateways, the gateway needs to provide a signature that is signed by the private key of the gateway id to remove the registration.
