A Client context for a ClientModule M

Client modules can interact with the whole client through this struct.

pub fn self_ref<'s>(&'s self) -> ClientContextSelfRef<'s, M>

Get a reference back to client module from the Self

It’s often necessary for a client module to “move self” by-value, especially due to async lifetimes issues. Clients usually work with &mut self, which can’t really work in such context.

Fortunately ClientContext is Clone and `Send, and can be used to recover the reference to the module at later time.
