pub async fn reissue<R: RngCore + CryptoRng>(
    &self,
    coins: Coins<SpendableCoin>,
    rng: R
) -> Result<OutPoint, ClientError>

Exchanges coins received from an untrusted third party for newly issued ones to prevent double spends. Users must ensure that the reissuance transaction is accepted before accepting coins as a valid payment.

On success the out point of the newly issued e-cash tokens is returned. It can be used to easily poll the transaction status using MintClient::fetch_coins until it returns Ok(()), indicating we received our newly issued e-cash tokens.
