Environment variable that specifies the bitcoind node. Necessary for LDK configuration if using bitcoind as the backend.

Environment variable that specifies the URL of an Esplora server. Necessary for LDK configuration if using esplora as the backend.
FM_LDK_NETWORK
Environment variable that specifies the Bitcoin network that the LDK Node should use. Must match FM_GATEWAY_NETWORK. Necessary for LDK configuration.
